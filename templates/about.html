Clinical Prediction Pipeline

The HealthGuard AI system models complex high-dimensional biomarker data leveraging a classic mathematical setup. Developed as a showcase for internship evaluation and clinical machine learning portfolios, it solves structured tabular diabetes predictions using a highly transparent workflow.

Machine Learning Workflow

  1. Dataset Acquisition: The system automatically verifies or acquires the Pima Indians Diabetes Dataset, compiling 768 patient historical data rows.
  2. Preprocessing & Cleaning: Biological invalid zeroes in crucial features (Glucose, Blood Pressure, Skin Thickness, Insulin, BMI) are replaced with median values.
  3. Standardization Scopes: Features are normalized using standard scaling (mean of 0, variance of 1) to optimize numerical regression convergence.
  4. Train-Test Splitting: The database is split (80% training set to fit statistical parameters, 20% validation set to confirm generalization capability).
  5. Parallel Training & Selection: Both a Logistic Regression and a Random Forest Classifier are trained, selecting the elite performer to save as model.pkl.
